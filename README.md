## Экскурсии в Сочи — лендинг

Одностраничный лендинг на **React + Vite + Tailwind CSS** для проекта «Честные экскурсии в Сочи и Абхазии».

### Установка

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

Откройте адрес из консоли (обычно `http://localhost:5173`) в браузере.

### Продакшен-сборка

```bash
npm run build
npm run preview
```
