## Экскурсии в Сочи — лендинг

Одностраничный лендинг на **React + Vite + Tailwind CSS** для проекта «Честные экскурсии в Сочи и Абхазии».

### Живая версия сайта

Основная ссылка для просмотра сайта:

- **Продакшен:** `https://excursions-localhost-landing.vercel.app`

### Установка

```bash
npm install
```

### Режим разработки (локально)

```bash
npm run dev
```

Откройте адрес из консоли (обычно `http://localhost:5173`) в браузере.  
Этот адрес доступен только на вашем компьютере во время работы локального сервера.

### Продакшен-сборка

```bash
npm run build
npm run preview
```
